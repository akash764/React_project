# React Project

## Prerequisites
- Node.js 18+ (recommended: latest LTS)
- npm (comes with Node.js)

## Install Dependencies
```bash
npm install
```

## Run in Development
```bash
npm run dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`).

## Build for Production
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```
